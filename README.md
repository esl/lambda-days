# Lambda Days

Static archive of **[lambdadays.org](https://lambdadays.org)** — the functional
programming conference for the Erlang, Elixir and wider FP community, held
annually in Kraków. The brand is retired; this repository preserves every past
edition as a self-contained static site served via GitHub Pages.

## Hosting

Static site, no build step. Enable GitHub Pages (branch `main`, root). The
[`CNAME`](CNAME) file points the site at `lambdadays.org`; once DNS is repointed
to GitHub Pages the original URLs (`lambdadays.org/lambdadays<year>`) resolve
against this archive.
